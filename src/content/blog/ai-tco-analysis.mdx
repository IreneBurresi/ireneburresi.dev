## Modello di Calcolo

```python
def calculate_tco(
    infrastructure_cost,
    personnel_cost,
    training_cost,
    maintenance_cost,
    years=3
):
    annual_cost = (
        infrastructure_cost +
        personnel_cost +
        training_cost +
        maintenance_cost
    )
    return annual_cost * years
```
