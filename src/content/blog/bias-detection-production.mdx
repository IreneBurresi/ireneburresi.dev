### Demographic Parity

```python
def demographic_parity(predictions, sensitive_attribute):
    groups = predictions.groupby(sensitive_attribute)
    positive_rates = groups.mean()

    # Calculate disparity
    max_rate = positive_rates.max()
    min_rate = positive_rates.min()

    disparity = max_rate - min_rate
    return disparity
```
